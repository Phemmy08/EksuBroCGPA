📖 Grading Information

🎯 EKSU Grading System (5.00 Scale)

Grade Points:
A = 5 (70-100)
B = 4 (60-69)
C = 3 (50-59)
D = 2 (45-49)
E = 1 (40-44)
F = 0 (0-39)

🏆 CGPA Classification

First Class: 4.50 - 5.00
Second Class Upper: 3.50 - 4.49
Second Class Lower: 2.40 - 3.49
Third Class: 1.50 - 2.39
Pass: 1.00 - 1.49

💡 How CGPA is Calculated

CGPA = Total Grade Points (TGP) ÷ Total Credit Units (TCU)
